# Tabbed Dashboard

A simple dashboard built with vanilla JavaScript that allows users to switch between different sections using tabs.

## Features 

- Tab-based navigation
- Dynamic content switching
- Clean and responsive UI
- Lightweight implementation without frameworks

## Technologies

- JavaScript
- HTML
- CSS

## What I learned

- DOM manipulation with JavaScript
- Event handling
- Structuring interactive UI without frameworks
- Managing UI state manually
